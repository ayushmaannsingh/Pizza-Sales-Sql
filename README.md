# Pizza-Sales-Sql
# 🍕 Pizza Sales SQL Project

A SQL-based analysis of a pizza restaurant's sales data, answering 13 business 
questions across three difficulty levels — Basic, Intermediate, and Advanced.

## 📌 About
This project uses SQL to explore order trends, revenue, and pizza performance 
using joins, aggregations, subqueries, and window functions (RANK, cumulative 
SUM) on a 4-table relational schema: `orders`, `order_details`, `pizzas`, and 
`pizza_types`.

## 🧠 Skills Demonstrated
- Joins (INNER JOIN across multiple tables)
- Aggregate functions (SUM, COUNT, AVG)
- Subqueries (scalar & correlated)
- Window functions (RANK() OVER, SUM() OVER)
- Data grouping & ranking for business insights

## 📂 Contents
- `queries.sql` — all 13 SQL queries
- `Pizza_Sales_SQL_QA.pdf` — Q&A document with explanations
- `Pizza_Sales_SQL_Project.pptx` — presentation deck
